# Py-Assistant
a little simple assistant made with python by NacreousDawn596 to kill boredom _(maybe it's usefull a little bit)_

# HOW TO INSTALL IT
```sh
sudo apt-get install git
git clone https://github.com/NacreousDawn596/Py-Assistant
cd Py-Assistant
chmod u+x setup.sh
./setup.sh
```

# HOW TO USE IT
```sh
python3 main.py
```
